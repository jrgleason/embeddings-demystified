# Image Example

## Grayscale Image as Vector

**Visual Example:**
- Grayscale image → Pixel values (0-255)
- Each pixel becomes a number in the vector
- Matrix representation converts to flat vector

**Limitations:**
- Sensitive to transformations (shifts, scaling, cropping)
- Great for maintaining pixel neighborhood information
- Need more sophisticated embeddings for robust image understanding

**Source:** https://ai.stanford.edu/~syyeung/
