# Example: King - Man + Woman = Queen

## How do we treat language like vector math?

### Vector Arithmetic Captures Relationships

```
king - man + woman ≈ queen
```

**Visual Representation:**
- Vectors for: king, man, woman, queen
- `king - man` = "royalty without gender"
- `+ woman` = "add female gender"
- Result ≈ `queen` (female royalty)

### What This Demonstrates:
- Embeddings capture semantic relationships
- Vector arithmetic reveals analogies
- Similar relationships = similar vector operations
- Famous Word2Vec example
